# Task25
Доработай свою многопоточную программу.
Реализуй в основном потоке (main thread) ожидание завершения выполнения второго потока.
Реализуй ограничение этого ожидания (например: 1 минута).
Реализуй возможность задавать величину ограничение ожидания из консоли.
После истечения этого времени ожидания первый поток должен вывести в консоль уведомление:”Больше ждать не буду” и прервать работу второго потока.
Обработай исключение которое бросит второй поток и выведи в консоль уведомление:”Выполнение задачи не завершено”
Строка вывода программы должна быть в формате:

имя потока: уведомление
